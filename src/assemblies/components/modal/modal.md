# 弹窗(modal)

 弹窗适用于：对页面内容作补充，与当前页面紧密相关，且从属于当前语境，相比二级页面，使用弹窗会让他们感觉更快捷、简单。
 <br>
 和气泡卡片相比，能承载更大信息量和更复杂的任务操作。在弹窗中可以嵌套任意信息，如：表格或表单。

## 基本用法
{{modal-base}}

## 其它所有用法
{{modal-all}}

## css修饰符

| 修饰符 | 说明 |
| ----- | ----- |
| xnomask | 去除弹窗摭罩层背景色 |

## 属性

| 属性名 | 说明 | 类型 | 可选值 | 默认值 |
| ----- | ----- | ----- | ----- | ----- |
| visible | 是否显示 | Boolean | / | false |
| draggable | 是否可拖动 | Boolean | / | true |
| weaking | 是否启用弱化样式 | Boolean | / | false |
