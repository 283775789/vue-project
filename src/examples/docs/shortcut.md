# 快捷输入解决以下问题

+ 将效率提升至极致：比从设计稿中取色，取尺寸等再写CSS出html再写JS，提高【20】倍以上的开发效率
+ 将规范化推行至极致: 快捷输入本质上就是完全相同，且经过多次验证的代码段

## 用法
1. 安装node平台的twui插件
<br>
<br>
```
npm install -g twui // 安装插件
twui shortcut // 下载快捷输入代码
```
